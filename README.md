# Misena
Cette application Android a été créée dans le cadre d'un projet final de cours Android.
Misena est une application simple et efficace de librairie video, elle tire ses informations de l'API TMDb.
Avec Misena, vous avez accès aux dernières tendances en termes de films et séries, ainsi que d'outils de recherche et de présentation des films et séries.

# Getting started
Pour installer cette application sur votre système, vous aurez besoin d'Android Studio (ou équivalent).

# Built with
Misena a été développée avec l'aide de Graddle, nous avons également utilisé Gson pour désérialiser les données reçues via l'API.
L'application comporte également des classes de requête sur l'API de TMDb.
